# Инструкция по использованию

##1.Сборка

### Собирать при помощи gradle.run на вкладке в IDE справа

##2.Конфигурация

### В открывшемся окне введём Ip-адрес, порт, который хотим занять и порт, который будем слушать. Нажмём StartNode

##3.Авторизация 

### Введём имя пользователя и нажмём Login. Перед нами откроется окно самого чата.

##4. Использование

### Введём сообщение и нажмём кнопку Send. Чтобы создать приватную группу, нажмём на имя пользователя слева.

### Если мы хотим создать ещё одного пользователя, снова запускаем приложение с помощью build.gradle (не останавливая работу приложения!),
### и при конфигурации в качестве прослушиваемого укажем порт, на котором висит первый пользователь.
